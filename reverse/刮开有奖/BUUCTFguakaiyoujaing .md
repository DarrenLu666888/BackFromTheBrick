说在前面：
这个题目有很多与WIN API　编程相关的知识，不过暂时没必要学的太深入；
收获：
1.明白了IDA上一些地址表示方式的具体意思，核心是：从地址思考，确定具体地址表示方式所代表的变量；
2.有些看不懂的程序代码可以直接把它整理写下来，特别是那些初始化处理
3.如果有密文，可以直接在那个MD5网站上搜，就可以看出是什么有名的加密，那样就不用自己解密了
